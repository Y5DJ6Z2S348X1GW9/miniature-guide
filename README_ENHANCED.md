## 🐛 **故障排除**

### ❓ **常见问题**

**Q: 游戏无法启动？**
A: 确保使用HTTP服务器，运行 `python start_game.py`

**Q: 没有音效？**
A: 点击游戏界面激活音频上下文，现代浏览器需要用户交互

**Q: 游戏卡顿？**
A: 在设置中降低粒子质量，关闭高级视觉效果

**Q: 资源丢失？**
A: 运行 `python scripts/generate_assets.py` 重新生成

**Q: 存档丢失？**
A: 检查浏览器本地存储，避免无痕模式

按 `Ctrl+Shift+D` 启用调试模式：

- 性能监控面板
- 对象计数器
- FPS显示
- 内存使用情况
